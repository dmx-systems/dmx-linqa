Modifying whiteboard contents

If you're an editor for the current workspace you can arrange the whiteboard items spatially, edit their text content, and modify their style.

You can move items by dragging them. The upper/left item corner will snap to the whiteboard grid to support alignment with other whiteboard items.

You can resize an item by clicking on it and then drag one of the 3 Resize handles. Use the mid-right handle to change width while adjusting height automatically so that the text content fits perfectly. Use the 2 bottom handles to adjust height (and width) manually for proper alignment with other whiteboard items.

You can rotate an item by dragging its Rotate handle.

Use the Edit mode to edit the original text content and to request another automatic translation. You also can edit the translation result manually.

Administrators can lock individual items so they can no longer modified by editors. For locked items editors (and administrators) see a lock symbol in the item's lower/right corner.
