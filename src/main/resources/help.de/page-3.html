Chat

Der Diskussionsbereich

Neben dem Whiteboard verfügt jeder Arbeitsbereich über einen Chat-/Diskussionsbereich. Sie können die Größe der Diskussion im Verhältnis zum Whiteboard einstellen. Schließen Sie den Diskussionsbereich, um dem Whiteboard vollen Raum zu geben.

Nachrichten und ihre Übersetzungen werden gleichzeitig angezeigt. Die Originalnachricht des Benutzers wird in der linken Spalte angezeigt, die Übersetzung in der rechten Spalte. Es wird angezeigt, ob die Übersetzung automatisch erstellt oder manuell bearbeitet wurde.

Die neuesten Nachrichten werden oben angezeigt. Für jede Nachricht werden das Datum und der Autor angezeigt. Wenn Sie den Mauszeiger auf den Autor setzen, wird Ihre E-Mail-Adresse angezeigt, sofern sie dies in Ihrem Benutzerprofil angegeben hat.

Schreiben Sie Ihre Nachricht in das Eingabefeld am unteren Rand. Formatieren Sie Ihre Nachricht, indem Sie Text auswählen. Dann erscheint ein Menü. Sie können Weblinks erstellen und Bilder einfügen. Sie können beliebige Dateien an Ihre Nachricht anhängen (Büroklammer-Button). Senden Sie Ihre Nachricht ab, indem Sie auf die Schaltfläche "Senden" (Papierflieger) klicken.

Sie können Ihre Nachricht mit einer anderen Nachricht verknüpfen. Sie können Ihre Nachrichten nachträglich bearbeiten und löschen.

Es gibt sowohl einen allgemeinen Chat als auch spezielle Diskussionen zu einem bestimmten Dokument oder Textblock.
