Ändern von Whiteboard-Inhalten

Wenn Sie ein Editor für den aktuellen Arbeitsbereich sind, können Sie die Whiteboard-Elemente räumlich anordnen, ihren Textinhalt bearbeiten und ihren Stil ändern.

Sie können Elemente verschieben, indem Sie sie ziehen. Die obere/linke Ecke des Elements rastet am Whiteboard-Raster ein, um die Ausrichtung mit anderen Whiteboard-Elementen zu unterstützen.

Sie können die Größe eines Elements ändern, indem Sie darauf klicken und dann einen der 3 Griffe für die Größenänderung ziehen. Verwenden Sie den mittleren rechten Griff, um die Breite zu ändern und gleichzeitig die Höhe automatisch anzupassen, damit der Textinhalt perfekt passt. Verwenden Sie die beiden unteren Griffe, um die Höhe (und Breite) manuell anzupassen, damit sie mit anderen Whiteboard-Elementen richtig ausgerichtet sind.

Sie können ein Element drehen, indem Sie an seinem Drehgriff ziehen.

Verwenden Sie den Bearbeitungsmodus, um den ursprünglichen Textinhalt zu bearbeiten und eine weitere automatische Übersetzung anzufordern. Sie können das Übersetzungsergebnis auch manuell bearbeiten.

Administratoren können einzelne Elemente sperren, so dass sie von Editoren nicht mehr geändert werden können. Bei gesperrten Elementen sehen sowohl Editoren als auch Administratoren ein Schlosssymbol in der unteren/rechten Ecke des Elements.
