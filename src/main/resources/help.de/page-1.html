Willkommen

Willkommen auf der Linqa-Kollaborationsplattform!

Die Linqa-Plattform hilft bei der Überwindung von Sprachbarrieren, indem sie gemeinsame Arbeitsbereiche bereitstellt. Ein Arbeitsbereich besteht sowohl aus einem Whiteboard, auf dem Texte, Dokumente, Videos, Notizen und Fotos platziert werden können, als auch aus einem Chat-/Diskussionsbereich.

Der Inhalt des Whiteboards und alle Chat-/Diskussionsnachrichten werden automatisch übersetzt. So ist alles, was auf der Plattform passiert, sofort in zwei Sprachen verfügbar. Diese zwei Sprachen können pro Linqa-Installation aus einem Vorrat von neun Sprachen konfiguriert werden.

Verwenden Sie das Icon oben rechts, um Ihre bevorzugte Anzeigesprache zu wählen.

Es gibt 3 Benutzerrollen: Mitglieder können Whiteboard-Inhalte lesen/sehen, chatten/diskutieren und Dokumente herunterladen. Editoren können Inhalte auf dem Whiteboard platzieren und anordnen. Administratoren können Arbeitsbereiche erstellen und Benutzer dazu einladen.
